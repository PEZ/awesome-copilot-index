# Awesome Copilot Index

An automatically generated index of the [awesome-copilot](https://github.com/github/awesome-copilot) repository, providing structured access to GitHub Copilot instructions, prompts, chat modes, and agents.

Visit the live index at: **[pez.github.io/awesome-copilot-index](https://pez.github.io/awesome-copilot-index/)**


## Powering the Awesome Copilot VS Code menu

[![Awesome Copilot VS Code Menu Demo](https://img.youtube.com/vi/AiL8LurZgSI/maxresdefault.jpg)](https://youtu.be/AiL8LurZgSI)

The index integrates with [Joyride](https://github.com/BetterThanTomorrow/joyride) to provide a seamless experience for accessing GitHub Copilot instructions, prompts, chat modes, and agents directly from VS Code.
